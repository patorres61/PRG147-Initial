Unit 1: JavaScript Basics

Initial JavaScript Program

This project required creating different divisions on a web page that use JavaScript coding. Requirements included buttons to make a division disappear and reappear, show date and time on a button click, show a greeting on a button click, and to change the color of a division.

Mathematics using JavaScript

This project required using JavaScript coding to compute mathematical equations. Requirements were to show examples for addition, subtraction, multiplication, division, division with a remainder, division with a rounded remainder, incrementing a number, and decrementing a number.

Unit 2:

JavaScript Functions

This project gave more practice in creating functions stored in a separate file. The goal was to create at least ten thumbnail images. When clicked on, the large image area would change to the selected image. Each image would display with a caption unique to that image.

JavaScript Events

JavaScript creates reactions to events on the webpage and different events can be programmed to trigger different reactions. Events include clicking, moving a mouse up, moving a mouse down while holding down the button, and moving the mouse away. Also, clicking, double-clicking or right-clicking may be used to trigger reactions. The requirements for this project were to create at least ten reactions to ten different events.
